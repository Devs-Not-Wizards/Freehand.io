Introduction

Want to create your own webpage design, just create it with pen, paper and your imagination.. "Name" helps you to create a HTML code for your webpage by just using a blueprint image of your design.

Predefined symbols are used to denote different components of the HTML page. The User has to just give a framework of the desired HTML page by drawing it on a blank paper using these symbols and uploading an image of the framework on the website. (Paint (Computer Application) can also be used to design the framework.) Requirements: Sketch Pen, Blank Paper or a image desiging tool.

Components:



To Display Image


To show Text


To create Button


To create Input View


Steps to generate Code



Requirements
  • A4 size Blank paper (non ruler).
  • Black sketch pen and measurement ruler.
  • Device with camara to capture image and upload it.

Designing the layout
  • Using the components defined in freehand.io, design your layout as per your idea.
  • Take the blank (non-ruler) page and draw the components.

Designing the layout
  • Consider exact relative size of the components used.
  • Do not overlap your components with other.
  • Use black sketch pen to draw your design.
  • The Shape for each symbols must clearly identifiable.
  • Do not write any text on the paper, you will be asked to input text at customization webpage.
  • Upload image in .jpg format only.
  • Try to Capture the image at perpendicular to the page.
